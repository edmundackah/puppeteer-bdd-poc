# Puppeteer BDD PoC

BDD wrapper for Puppeteer

## Features
- Environment specific configuration
- Lighthouse User Flow Reports
- Video Recorder
- Puppeteer User Flow JSON support
- Screenshot
- Custom HTML report
- Parallel exceution
- Rerun failed tests
- Device Info embedded in report
- CI report generator (JUnit)
- Centralised Logging

## Installation

Install project with npm
```bash
  npm install
```

## Running Tests

To run tests, run the following command
```bash
  npm test
```

Retry failed tests
```bash
  npm run test:failed
```

Run cleanup script
```
  npm run cleanup
```
